## Project Overview
This artwork was selected because of its strong atmospheric qualities, layered spatial composition, glowing biological forms, and ecosystem-like structure, which naturally support interactive reinterpretation.

---

# Mechanic Breakdown

## 1. User Input Mechanic

### Concept
The viewer becomes an active force inside the ecosystem.

### Features
- nearby particles react to mouse movement
- organisms glow or scatter when approached
- clicking generates ripple effects and energy blooms
- dragging creates movement paths through the environment
- keyboard input changes interaction states

### Purpose
This mechanic transforms the audience from passive observers into participants within the artwork.

---

## 2. Time-Based Mechanic

### Concept
The ecosystem evolves gradually over time.

### Features
- environmental brightness changes slowly
- particles fade in and out rhythmically
- organisms grow and decay
- different areas of the landscape awaken over time
- colour transitions create changing atmosphere

### Purpose
This mechanic gives the world a sense of life, progression, and environmental transformation.

---

## 3. Perlin Noise & Randomness Mechanic

### Concept
Organic motion is generated procedurally using Perlin noise and randomness.

### Features
- floating particle movement
- drifting environmental motion
- subtle breathing animations
- unstable glow distortion
- random biological growth patterns

### Purpose
This mechanic makes the ecosystem feel natural, alive, and unpredictable.

---

## 4. Audio Mechanic

### Concept
The ecosystem reacts dynamically to sound.

### Features
- glow intensity responds to music amplitude
- bass frequencies affect large environmental structures
- high frequencies activate smaller particles
- audio pulses create ripple-like visual movement
- sound changes the ecosystem’s overall energy level

### Purpose
This mechanic strengthens immersion and audiovisual interaction.

---

# Why This Artwork Was Selected

The selected artwork is highly suitable for creative coding reinterpretation because:

- it already contains ecosystem-like structures
- it suggests movement and life
- it supports layered interactions
- it allows multiple mechanics to coexist naturally
- it can be translated into a generative system rather than a static copy

This makes the project more cohesive and interactive compared to traditional artwork recreations.
